# Swap & Shop 🛍️

An online e-commerce web app where users can buy, sell, or swap items.

## 🚀 Features
- Product listing UI
- Watchlist dropdown
- Reusable components
- Responsive design

## 🛠️ Tech Stack
- React (Vite)
- Tailwind CSS
- JavaScript

## 📦 Getting Started

```bash
npm install
npm run dev
